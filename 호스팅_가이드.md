# 🍀 로또 번호 생성기 — GitHub Pages 무료 호스팅 가이드

닷홈처럼 접속이 끊기지 않고, **무료 · 영구 · HTTPS**로 운영하는 방법입니다.
명령어(Git)를 몰라도 **github.com 웹사이트에서 마우스만으로** 전부 할 수 있어요.

---

## 1단계. GitHub 가입
1. https://github.com 접속 → **Sign up** 으로 계정 생성 (무료)

## 2단계. 저장소(폴더) 만들기
1. 오른쪽 위 **+** → **New repository**
2. **Repository name**: `lotto` (원하는 이름)
3. **Public** 선택
4. **Create repository** 클릭

## 3단계. 파일 올리기
1. 저장소 화면에서 **Add file → Upload files**
2. 이 폴더의 파일들을 드래그해서 올립니다:
   - `index.html`, `app.js`, `styles.css`, `icon.svg`
   - `data` 폴더 (`draws.json` 포함)
3. 아래 **Commit changes** 클릭

## 4단계. Pages 켜기 (사이트 주소 발급)
1. 저장소 상단 **Settings** → 왼쪽 메뉴 **Pages**
2. **Source** = `Deploy from a branch`
3. **Branch** = `main`, 폴더 = `/ (root)` → **Save**
4. 1~2분 뒤 상단에 주소가 나타납니다:
   `https://<내아이디>.github.io/lotto/`
   → 이 주소가 **아버지께 알려드릴 고정 주소**입니다.

> 끝입니다! 복잡한 설정(자동화·권한) 없이 이대로 잘 작동합니다.

---

## 📱 아버지 휴대폰 홈화면에 추가하기
- **아이폰(Safari)**: 주소 접속 → 아래 **공유 버튼** → **홈 화면에 추가**
- **안드로이드(Chrome)**: 주소 접속 → 오른쪽 위 **⋮** → **홈 화면에 추가**
- 그러면 앱처럼 🍀 아이콘으로 바로 열 수 있습니다.

---

## 🎯 당첨 확인 사용법
- 앱 하단 **[🎯 당첨 확인]** 버튼을 누르면 됩니다.
- **최신 당첨번호가 자동으로 불러와집니다.** 회차를 골라서 내가 뽑아둔 번호와
  **자동 대조해 등수(1~5등/낙첨)** 를 보여주고, 맞은 번호는 초록 테두리로 표시합니다.
- 회차 드롭다운에서 최근 6회차 중 원하는 회차를 선택할 수 있어요.

> (참고) 당첨번호는 공개 데이터(github.com/smok95/lotto, 매주 자동 갱신)에서
> 불러옵니다. 인터넷이 안 되거나 데이터가 늦을 때를 대비해
> **"당첨번호 직접 입력"** 칸도 그대로 있습니다.

---

## 🔄 나중에 내용 수정하려면
1. 저장소에서 수정할 파일 클릭 → 연필(✏️) 아이콘 → 수정 → **Commit changes**
2. 또는 **Add file → Upload files** 로 새 파일 덮어쓰기
3. 1~2분 뒤 사이트에 자동 반영
